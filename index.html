About the Project

The Smart Weather Monitoring & Public Awareness Rover was developed in response to the growing concerns about air Weather in urban areas and the need for localized, real-time monitoring and public alerts.

Urban air Weather is responsible for millions of premature deaths globally each year. While fixed monitoring stations provide valuable data, they can't capture the microclimate variations that occur within cities at street level where people are most exposed.

Our mobile rover solution addresses this gap by offering dynamic monitoring capabilities that can navigate through urban spaces, detecting Weather hotspots and providing immediate feedback to nearby pedestrians through visual displays and audio alerts.

Technical Overview

Components Used

  • Arduino Uno (Main controller)
  • ESP8266 WiFi Module (Data transmission)
  • DHT11 Sensor (Temperature & humidity)
  • MQ135 Sensor (Air quality)
  • 16x2 LCD Display (Local readings)
  • Bluetooth Module HC-05 (Remote control)
  • Bluetooth Speaker (Public alerts)
  • DC Motors & Motor Driver (Movement)
  • LiPo Battery Pack (Power supply)
  • 3D Printed Chassis (Structure)

System Architecture

The system follows a modular architecture with these key components:

  • Sensing Module: Collects environmental data
  • Processing Unit: Analyzes sensor data
  • Control Module: Handles rover movement
  • Communication Module: Manages data transfer
  • Alert System: Triggers warnings when necessary
  • Power Management: Optimizes battery usage

The Arduino serves as the central brain, processing sensor inputs and controlling outputs, while the ESP8266 handles wireless communication for data logging and remote access.
